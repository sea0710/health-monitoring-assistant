# 愈见·血常规监测助手

## 项目简介

**愈见·血常规监测助手**是一款专为肿瘤患者及家属设计的医疗健康监测工具，实现了「上传报告→OCR识别→异常解读→趋势查看」的核心业务流程，帮助用户自主监测血常规指标变化，建立对健康状况的掌控感。

## 核心功能

### 1. 报告上传与OCR识别
- 支持相机拍摄或相册选择血常规报告图片
- 调用OCR云函数自动识别报告数据
- 支持手动修正识别结果

### 2. 异常分级预警
- 基于医疗标准的四级异常分级（正常、Ⅰ度、Ⅱ-Ⅲ度、Ⅳ度）
- 智能标记异常指标，提供视觉预警

### 3. 趋势分析
- 核心指标历史变化趋势图
- 参考区间标注与异常点标记
- 极值标注与数据点交互

### 4. 智能提醒
- 基于化疗结束时间的复查提醒设置
- 一键设置复查提醒（+6天、+13天）
- 待提醒列表与历史记录管理

### 5. 患者档案管理
- 患者基本信息管理
- 多患者切换支持

## 技术架构

### 前端
- **微信小程序**：原生小程序开发
- **UI框架**：自定义组件 + Tailwind CSS
- **图表库**：ECharts
- **状态管理**：全局数据存储

### 后端
- **服务端**：NestJS (TypeScript)
- **云函数**：微信云开发
- **OCR服务**：AI图像识别
- **数据存储**：本地存储 + 云数据库

### 技术栈
- **前端**：JavaScript, Tailwind CSS, ECharts
- **后端**：TypeScript, NestJS, 微信云开发
- **工具**：ESLint, Prettier, Git

## 安装与运行

### 微信小程序端
1. **前提条件**：
   - 微信开发者工具
   - 微信云开发环境

2. **安装步骤**：
   - 克隆项目到本地
   - 打开微信开发者工具，导入项目
   - 配置云开发环境ID
   - 部署云函数

3. **运行**：
   - 在微信开发者工具中预览
   - 或扫码在微信中体验

### 后端服务
1. **前提条件**：
   - Node.js 16+
   - npm 或 yarn

2. **安装步骤**：
   ```bash
   cd server
   npm install
   ```

3. **运行**：
   ```bash
   npm run start:dev
   ```

## 核心API

### 云函数API
- `ocr`：识别血常规报告图片
- `report`：报告数据管理
- `login`：用户认证
- `checkReminders`：检查并发送提醒

### 后端API
- `GET /api/reports`：获取报告列表
- `GET /api/reports/:id`：获取报告详情
- `POST /api/reports`：创建新报告
- `PUT /api/reports/:id`：更新报告
- `DELETE /api/reports/:id`：删除报告

### 视觉风格
- **主色调**：蓝绿色（医疗专业感+生命活力）
- **布局**：大圆角卡片，温暖米灰背景
