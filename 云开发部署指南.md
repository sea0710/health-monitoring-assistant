### 第二步：开通云开发

1. 在开发者工具中，点击工具栏上的"云开发"按钮
2. 首次使用需要开通云开发服务

### 第三步：创建云数据库集合

在云开发控制台中，创建以下数据库集合：

1. **users** - 用户表
   - 字段：phone, password, created_at, updated_at

2. **patients** - 患者表
   - 字段：user_id, name, gender, birthday, tumor_type, created_at, updated_at

3. **reports** - 报告表
   - 字段：user_id, patient_id, test_time, test_hospital, abnormal_count, created_at, updated_at

4. **indicators** - 指标表
   - 字段：report_id, indicator_code, indicator_name, test_value, test_unit, level, created_at

### 第六步：修改小程序代码

在小程序代码中，将所有API调用改为云函数调用：

**示例**：
```javascript
// 原来的方式
wx.request({
  url: 'https://api.yourdomain.com/api/login',
  method: 'POST',
  data: { phone, password },
  success: (res) => {
    // 处理结果
  }
})

// 改为云函数调用
wx.cloud.callFunction({
  name: 'login',
  data: { phone, password },
  success: (res) => {
    // 处理结果
  }
})
```

## 🔧 常见问题

### 1. 云函数部署失败
- 检查package.json是否正确
- 检查依赖是否完整
- 查看云函数日志

### 2. 数据库权限问题
- 在云开发控制台中设置数据库权限
- 建议设置为"仅创建者可写，所有人可读"

### 3. OCR识别失败
- 检查环境变量是否配置正确
- 检查API密钥是否有效
- 查看云函数日志
