# 指标单位统一与换算方案

## 一、问题描述

不同医院的血常规检测报告使用不同的单位，需要统一换算标准以便进行准确的分级评估。

## 二、单位换算规则

### 1. 中性粒细胞绝对值（NE#/NEUT#）
- **常见单位**：
  - ×10⁹/L（国内主流）
  - ×10³/μL（部分医院/国外）
- **换算关系**：1 ×10⁹/L = 1 ×10³/μL（数值完全一致，无需换算）
- **统一解读单位**：×10⁹/L

### 2. 血小板计数（PLT）
- **常见单位**：
  - ×10⁹/L（国内主流）
  - ×10³/μL（部分医院）
  - /μL（部分医院）
- **换算关系**：
  - 1 ×10⁹/L = 1 ×10³/μL（数值一致，无需换算）
  - 1 ×10⁹/L = 1000 /μL（需要换算）
- **统一解读单位**：×10⁹/L

### 3. 血红蛋白（Hb/HGB）
- **常见单位**：
  - g/L（国内主流）
  - g/dL（部分医院/国外）
- **换算关系**：1 g/dL = 10 g/L（例：12 g/dL = 120 g/L）
- **统一解读单位**：g/L

### 4. 白细胞总数（WBC）
- **常见单位**：
  - ×10⁹/L（国内主流）
  - ×10³/μL（部分医院/国外）
- **换算关系**：1 ×10⁹/L = 1 ×10³/μL（数值完全一致，无需换算）
- **统一解读单位**：×10⁹/L

### 1. 单位换算服务 API

```javascript
// 单位换算服务
const UnitConversionService = {
  /**
   * 换算指标数值到标准单位
   * @param {string} indicatorCode - 指标代码（WBC, NEUT#, HGB, PLT）
   * @param {number} value - 原始数值
   * @param {string} unit - 原始单位
   * @returns {Object} - { standardizedValue, standardUnit, conversionFactor }
   */
  convertToStandard: (indicatorCode, value, unit),
  
  /**
   * 识别单位类型
   * @param {string} unit - 单位字符串
   * @returns {string} - 标准化单位代码
   */
  identifyUnit: (unit),
  
  /**
   * 验证数值合理性
   * @param {string} indicatorCode - 指标代码
   * @param {number} value - 数值
   * @param {string} unit - 单位
   * @returns {boolean} - 是否合理
   */
  validateValue: (indicatorCode, value, unit)
}
```
