# 页面优化计划

## 任务概述
优化分级速查、报告详情、趋势分析3个页面，按照新的前端样式和内容更新。

## 具体任务

### 任务1：分级速查页面优化

**目标**：根据新设计稿更新分级速查页面的样式和内容
**步骤**:
1. 分析新设计稿 `GradeReferencePage.tsx` 的样式结构
2. 提取完整的分级数据（包括 ANC, NEUT#, PLT, HGB）
3. 更新后端数据结构
4. 更新前端 HTML 和 CSS 样式
5. 更新分级速查页面的显示逻辑
6. 添加出院医嘱弹窗功能
7. 测试验证功能
**预计工作量**: 2-3小时
**优先级**: 高
**依赖**: 新设计稿文件

---

**目标**：根据新设计稿更新报告详情页面的样式和内容
