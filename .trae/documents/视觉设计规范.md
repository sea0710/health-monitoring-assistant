# 愈见·血常规监测助手 - 视觉设计规范

## 📋 设计原则

基于品牌定位（温暖、可靠、简洁）和用户需求（安心、信任、掌控感），制定以下设计原则：

1. **温暖而不失专业** - 圆润元素传达温暖，清晰数据体现专业
2. **简洁而不简陋** - 精简信息，通过留白创造高级感
3. **清晰而不拥挤** - 明确的视觉层级，充足的间距
4. **一致而不单调** - 统一的设计语言，适当的视觉变化
5. **易用而不复杂** - 简化操作，清晰的视觉反馈

## 🎬 动画规范

### 过渡动画
```css
--transition-fast: 0.15s ease;    /* 快速 - 按钮点击 */
--transition-base: 0.2s ease;     /* 基础 - 状态变化 */
--transition-slow: 0.3s ease;     /* 慢速 - 页面切换 */
```

### 缓动函数
```css
--ease-in-out: ease;              /* 标准缓动 */
--ease-out: cubic-bezier(0.4, 0, 0.2, 1);  /* 减速缓动 */
--ease-in: cubic-bezier(0.4, 0, 1, 1);     /* 加速缓动 */
```

### 动画原则
- **简洁**：避免过度动画，仅在必要时使用
- **自然**：使用标准缓动函数，避免弹跳效果
- **快速**：动画时长控制在200ms以内，保证响应速度
- **有意义**：动画应传达状态变化，而非单纯装饰
