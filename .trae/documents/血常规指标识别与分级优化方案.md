# 血常规指标识别与分级优化方案

## 一、需求概述

根据用户提供的权威标准，优化血常规指标识别规则、单位统一规则和临床分级阈值，确保系统能够准确识别不同医院、不同格式的报告，并提供专业的分级评估和处理建议。

## 二、当前实现与需求对比

### 1. 指标识别规则对比

#### 当前支持的格式：
- **白细胞计数**：`白细胞(WBC)`、`白细胞计数(WBC)`、`WBC`
- **中性粒细胞绝对值**：`中性粒细胞# (NE#)`、`中性粒细胞绝对值(Neu#)`、`NE#`
- **血红蛋白**：`血红蛋白(HGB)`、`HGB`、`Hb`
- **血小板计数**：`血小板(PLT)`、`血小板计数(PLT)`、`PLT`

#### 需要新增的格式：
- **白细胞计数**：`白细胞`（不带括号）
- **中性粒细胞绝对值**：`中性粒细胞#`、`NEUT#`
- **血红蛋白**：`血色素`
- **血小板计数**：`血小板`（不带括号）

### 2. 单位统一规则对比

#### 当前实现：
- ✅ 白细胞/中性粒细胞/血小板：`×10⁹/L` = `×10³/μL`（数值等价）
- ✅ 血红蛋白：`g/L` = `g/dL × 10`

#### 需要优化：
- ✅ 已正确实现，无需修改

### 3. 临床分级阈值对比

#### 当前实现的问题：

**中性粒细胞绝对值（×10⁹/L）**：
- 当前：正常 ≥2.0，Ⅰ度 1.5-2.0，Ⅱ度 1.0-1.5，Ⅲ度 0.5-1.0，Ⅳ度 <0.5
- 需求：正常 2.0-7.5，Ⅰ度 1.5-2.0，Ⅱ度 1.0-1.5，Ⅲ度 0.5-1.0，Ⅳ度 <0.5
- **差异**：需要增加上限判断（>7.5 为偏高，不标记骨髓抑制）

**血小板计数（×10⁹/L）**：
- 当前：正常 ≥100，Ⅰ度 75-100，Ⅱ度 50-75，Ⅲ度 25-50，Ⅳ度 <25
- 需求：正常 100-300，Ⅰ度 75-100，Ⅱ度 50-75，Ⅲ度 25-50，Ⅳ度 <25
- **差异**：需要增加上限判断（>300 为偏高，不标记骨髓抑制）

**血红蛋白（g/L）**：
- 当前：正常 ≥120，Ⅰ度 100-120，Ⅱ度 80-100，Ⅲ度 60-80，Ⅳ度 <60
- 需求：正常 男 120-160 / 女 110-150，Ⅰ度 男 100-120 / 女 90-110，Ⅱ度 80-100，Ⅲ度 60-80，Ⅳ度 <60
- **差异**：需要支持性别区分，增加上限判断

### 4. 输出要求对比

#### 当前实现：
- ✅ 显示指标、数值、单位、分级
- ❌ 缺少整体风险结论
- ❌ 缺少对应处理原则
- ❌ 缺少复查时间与警惕症状提示

#### 需要新增：
- 整体风险结论（如"存在Ⅱ度骨髓抑制风险"）
- 对应处理原则（如"建议就医，遵医嘱处理"）
- 复查时间提示（如"建议3-5天后复查"）
- 警惕症状提示（如"如出现发热、出血等症状，请立即就医"）

#### 任务 3：更新分级计算逻辑
**文件**：`server/src/services/gradeService.js`

**修改内容**：
```javascript
const calculateGrade = (indicatorCode, testValue, referenceMin, referenceMax, gender = 'male') => {
  const value = parseFloat(testValue)
  
  if (isNaN(value)) {
    return { level: 'normal', is_abnormal: false }
  }
  
  const refMin = parseFloat(referenceMin)
  const refMax = parseFloat(referenceMax)
  
  // 判断是否异常
  const isAbnormal = !isNaN(refMin) && !isNaN(refMax) && (value < refMin || value > refMax)
  
  if (!isAbnormal) {
    return { level: 'normal', is_abnormal: false }
  }
  
  // 特殊处理血红蛋白（需要性别）
  if (indicatorCode === 'HGB') {
    const rules = GRADE_RULES.HGB[gender] || GRADE_RULES.HGB.male
    
    if (value > rules.normal.max) {
      return { level: 'normal', is_abnormal: false, note: '指标偏高' }
    }
    
    if (value < rules.IV.max) return { level: 'IV', is_abnormal: true }
    if (value < rules.III.max) return { level: 'III', is_abnormal: true }
    if (value < rules.II.max) return { level: 'II', is_abnormal: true }
    if (value < rules.I.max) return { level: 'I', is_abnormal: true }
    
    return { level: 'normal', is_abnormal: false }
  }
  
  // 其他指标
  const rules = GRADE_RULES[indicatorCode]
  if (!rules) {
    return { level: 'normal', is_abnormal: false, note: '指标异常' }
  }
  
  // 指标升高不标记骨髓抑制
  if (value > rules.normal.max) {
    return { level: 'normal', is_abnormal: false, note: '指标偏高' }
  }
  
  // 骨髓抑制分级
  if (value < rules.IV.max) return { level: 'IV', is_abnormal: true }
  if (value < rules.III.max) return { level: 'III', is_abnormal: true }
  if (value < rules.II.max) return { level: 'II', is_abnormal: true }
  if (value < rules.I.max) return { level: 'I', is_abnormal: true }
  
  return { level: 'normal', is_abnormal: false }
}
```

```javascript
const generateOverallRisk = (indicators) => {
  const abnormalIndicators = indicators.filter(i => i.is_abnormal)
  
  if (abnormalIndicators.length === 0) {
    return {
      level: 'normal',
      conclusion: '各项指标正常，未见明显异常',
      severity: 0
    }
  }
  
  const maxLevel = Math.max(...abnormalIndicators.map(i => {
    const levelMap = { 'I': 1, 'II': 2, 'III': 3, 'IV': 4 }
    return levelMap[i.level] || 0
  }))
  
  const levelNames = { 1: 'Ⅰ度', 2: 'Ⅱ度', 3: 'Ⅲ度', 4: 'Ⅳ度' }
  const levelSeverity = { 1: '轻度', 2: '中度', 3: '重度', 4: '极重度' }
  
  return {
    level: levelNames[maxLevel],
    conclusion: `存在${levelNames[maxLevel]}骨髓抑制风险`,
    severity: maxLevel,
    description: levelSeverity[maxLevel]
  }
}
```
